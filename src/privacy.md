# Privacy

Questo sito è un sito statico informativo del condominio Via Casilina 9.

Il sito non usa analytics, non usa cookie di profilazione, non contiene moduli di contatto e non raccoglie dati personali tramite database o backend applicativi.

I contenuti sono pubblicati come pagine statiche e ospitati tramite GitHub Pages. GitHub può trattare dati tecnici necessari all'erogazione del servizio, come indicato nella documentazione e nelle informative privacy di GitHub.

Per mantenere il sito semplice e rispettoso della privacy, non devono essere aggiunti strumenti di tracciamento, risorse esterne invasive o documenti con dati personali reali senza una valutazione preventiva.

## Richieste di abilitazione all'area riservata

Per richiedere l'accesso all'area documenti riservati è disponibile un modulo Google Forms collegato dalla pagina "Area riservata".

Il modulo può richiedere dati come nome, cognome, email da abilitare, telefono, interno, scala o edificio, piano, qualifica rispetto all'unità immobiliare e eventuali note. Questi dati sono usati esclusivamente per verificare la richiesta di abilitazione all'area riservata.

La compilazione del modulo non comporta accesso automatico. Ogni richiesta viene verificata manualmente dall'amministrazione o dal referente incaricato. Se la richiesta viene approvata, l'email indicata potrà essere aggiunta alla lista degli utenti autorizzati su Cloudflare Access.

L'area riservata è gestita separatamente tramite Cloudflare Access. L'accesso avviene tramite codice temporaneo inviato all'indirizzo email autorizzato.

Non inviare tramite il modulo dati non richiesti, documenti di identità, codici fiscali, dati sanitari o altre informazioni non necessarie.

Per richieste di chiarimento, aggiornamento o cancellazione dei dati inviati tramite il modulo, è possibile scrivere a [postmaster@viacasilina9.it](mailto:postmaster@viacasilina9.it).
